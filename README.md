# PhoneGap DataWedge Intent
===========================

Zebra BarcodeScanner for Cordova / PhoneGap.

## Installation

    
This requires phonegap 5.0+ ( current stable v3.0.0 )

    phonegap plugin add https://github.com/conveyGmbH/phonegap-datawedge-intent.git

### Supported Platforms

- Android
